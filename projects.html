FinMesh

A Python API wrapper for various financial APIs.



FinMesh is a python-based package that brings together financial data from various sources in one place for ease of use and distribution. The four main sections of FinMesh are (1) the IEX REST API, (2) data from the US treasury, (3) data from the US Federal Reserve Economic Data, and (4) data from the SECs EDGAR system.

I have written all about this project in multiple locations, so I'll spare the details here.

If you can read all about FinMesh by clicking the title above, or by clicking the link in the navigation bar which will bring you to the ReadTheDocs page for FinMesh.

You can `pip install FinMesh` if you want to try it out for yourself.

Website Element Painter

A small Python add-on for people that manage their own HTML-based websites.



I was tired of making sure all my head, header, and footer elements matched across web pages. When I started searching for solutions I found plenty of people that had come up on this issue but not many that offered a satisfactory solution. Turns out that with vanilla HTML and CSS, what I was looking for was impossible.

There are options - yes. You have PHP, but that is server side and with my website being hosted on GitHub for now, that's not an option. Javascript and html_include is probably the best option out there for this, and it is probably better than this solution. That said, Javascript adds weight and reliability issues that I'm not sure I want to deal with.

So I wrote this little script to automate the update of headers, footers, and heads. It is dead simple, and relies on a human marking each of their pages with HTML comments. The thing is, it's simple to understand, all the processing is done locally with the developer (me), so mistakes are caught immediately and before they are published, and all my webpages remain plain HTML and CSS for maximum speed.

You can read a little more and view the code by clicking the title above.

EDGAR Filings Scraper

A Node.js scraper for the SEC EDGAR database.



I originally created a website scraper as part of an Ania Kubow tutorial, and was wanting to customize and extend the functionality a bit. I didn't find many examples on services like Rapid API so I figured I would give it a go. I had already worked with the EDGAR system extensively in Python so I was familiar with it.

This API supports gathering (1) the most recent filings for all US-based companies, (2) the most recent filings for a particular filing organization, and (3) the most recent filings of a certain type for a particular filing organization.
